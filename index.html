How dose JavaScript Work?

JavaScript is a client-side scripting and interpreted language. Client-side scripting language means that it runs at the client-side inside the web-browsers. When the browser loads the page, An interpreter can take the raw JavaScript code and run it. So JavaScript is an interpreted language. Interpreted language means it does not need a compiler to translate its code. The browser has a built-in interpreter. Some populer interpreter is V8(chrome), spidermonkey(firefox).

Different Between local storage and Session storage?

Local and Session storage are the web storage objects and they have the same API. LocalStorage persists the storage across browser sessions. So if the browser is closed or reopened, the storage is saved. Thats Why Local storage data has no expiration date. When open a new tab or a new window, Session storage creates a new session and new storage. Session storage are valid for only one tab at a time and when the browser is closed or reopened, the Session storage is expira.
